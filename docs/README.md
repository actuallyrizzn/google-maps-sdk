## Overview

This documentation was generated by scraping the official Google Maps Platform documentation using browser automation. It covers:

- **Routes API**: Modern routing API with traffic-aware routing
- **Directions API (Legacy)**: Legacy directions service
- **Roads API**: Road identification, snapping, and speed limits
- **Traffic Layer**: JavaScript API for displaying traffic conditions

## Quick Reference

### Routes API
- **Base URL**: `https://routes.googleapis.com`
- **Endpoints**: 2 REST endpoints
- **Status**: Active (Recommended)

### Directions API (Legacy)
- **Base URL**: `https://maps.googleapis.com/maps/api/directions`
- **Endpoints**: 2 output formats (JSON, XML)
- **Status**: Legacy (Consider migrating to Routes API)

### Roads API
- **Base URL**: `https://roads.googleapis.com/v1`
- **Endpoints**: 3 endpoints
- **Status**: Active

### Traffic Layer
- **Type**: JavaScript API (not REST)
- **Status**: Active
